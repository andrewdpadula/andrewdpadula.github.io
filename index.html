Projects

Virtual Pet Shelter

An interactive text-based game written in Java. Users are asked to care for virtual pets that are stored in a shelter. Hunger, thirst, and boredom are measured for each pet and alleviated by feeding, watering, and playing, respectively. The player is able to adopt pets, removing them from the shelter. The player can also add pets to the shelter, giving them a name and character trait.


Skills used: Java, TDD.

Virtual Pets Amok

Much like "Virtual Pet Shelter" but with several twists and a greater emphasis on OOP principles, especially encapsulation and inheritance. This game introduces robotic pets to go with the traditional organic pets. The pets now have additional needs, including for organic pets a need to go to the bathroom and for robotic pets a need to be oiled and maintained. Dogs can be walked and their cages can be cleaned, while cats have a communal litter box that requires cleaning.


Skills used: Java, TDD.
